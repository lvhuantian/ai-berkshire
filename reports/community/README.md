# 社区研究报告 / Community Reports

这里收录社区成员**用本仓库 skill 产出**的研究报告，按 `[公司名]/` 组织。提交要求见 [CONTRIBUTING.md](../../CONTRIBUTING.md)：文首注明所用 skill 和模型、附免责声明、一个 PR 一份报告。

本目录报告由社区贡献者产出，不代表仓库维护者的观点，也不构成投资建议。维护者自己的研究在 `reports/` 其他目录。

Reports here are produced by community members **using this repo's skills**, organized as `[company]/`. See [CONTRIBUTING.md](../../CONTRIBUTING.md) for requirements: state the skill and model used at the top, include a disclaimer, one report per PR.

Reports in this directory reflect their contributors' output, not the maintainer's views, and are not investment advice. The maintainer's own research lives in the other `reports/` directories.
